# Portfolio — Tên Bạn

Trang portfolio đơn giản (HTML/CSS/JS) được tạo để demo.

Mô tả ngắn:
- `index.html` — trang chính
- `styles.css` — style toàn bộ giao diện
- `script.js` — thao tác: theme, form demo, smooth scroll

Mở trang (Windows PowerShell):

```powershell
# Mở file index.html bằng trình duyệt mặc định
Invoke-Item .\\index.html
```

Gợi ý:
- Thay `Tên Bạn` bằng tên thật của bạn.
- Thay ảnh bằng ảnh của bạn (thêm vào thư mục `assets/` và sửa src).
- Để gửi form thực sự, kết nối `script.js` tới một API backend.

Chúc bạn có trang portfolio đẹp!
